Date

Converts between locale specific date string and Date value.

Example 1: Default converter on the input date component (when converter option is not set).

Example 2: converter option is set to - options: {year: 'numeric', month: 'long', day: 'numeric'}

Example 3: converter option is set to - options: {formatType: 'date', dateFormat: 'medium'}

Date and Time

Converts between locale specific date and time string and Date value.

Example 1: Default converter on the input dateTime component (when converter option is not set).

Example 2: converter option is set to - options : {formatType: 'datetime', dateFormat: 'short', timeFormat: 'short'}.

Example 3: converter option is set to - options : {pattern: 'MMM d, y, h:mm:ss a'}

Time

Converts between locale specific time string and Date value.

Example 1: Default converter on the input time component (when converter option is not set).

Example 2: converter option set to - options : {formatType: 'time', timeFormat: 'full'}.

Example 3: converter option set to - options : {hour: '2-digit', minute: '2-digit', second: '2-digit'}.
